# Чек-лист вопросов для архитектора (личный бренд)

Данный документ поможет собрать всю необходимую информацию для разработки персонального сайта архитектора.

---

## 1. О вас

- [ ] Полное имя (как будет отображаться на сайте)
- [ ] Творческий псевдоним / бренд-нейм (если есть)
- [ ] Город и страна проживания
- [ ] Образование (вуз, факультет, год выпуска)
- [ ] Дополнительное образование, курсы, стажировки за рубежом
- [ ] Стаж в архитектуре (сколько лет практики)
- [ ] Основная специализация (жилая архитектура, интерьеры, общественные здания, ландшафт и т.д.)
- [ ] Краткая биография (2-3 предложения — как вы представляете себя клиентам)
- [ ] Развёрнутая история: как пришли в архитектуру, что вас вдохновляет
- [ ] Членство в профессиональных союзах / организациях
- [ ] Личные награды и достижения
- [ ] Публикации в СМИ / профессиональных изданиях
- [ ] Выступления, лекции, мастер-классы

---

## 2. Позиционирование и личный бренд

### Целевая аудитория
- [ ] Кто ваш идеальный клиент? (частные заказчики, застройщики, владельцы бизнеса, рестораторы)
- [ ] Ценовой сегмент (средний, премиум, люкс)
- [ ] География клиентов (город, страна, международный уровень)
- [ ] Типичный запрос клиента, который к вам обращается

### Ваша уникальность
- [ ] Чем вы отличаетесь от других архитекторов?
- [ ] Какие 3 слова лучше всего описывают ваш стиль?
- [ ] Ваша архитектурная философия / манифест (в 1-2 предложениях)
- [ ] Есть ли авторский подход или методология, которую вы применяете?
- [ ] Какую эмоцию / ощущение должен вызывать ваш сайт у посетителя?

### Тон коммуникации
- [ ] Какой стиль общения вам ближе? (строгий профессиональный / дружелюбный экспертный / творческий-вдохновляющий)
- [ ] Обращение к клиентам на «ты» или на «вы»?
- [ ] Повествование от первого лица («Я проектирую...») или от третьего («Архитектор создаёт...»)?

### Конкуренты / ориентиры
- [ ] Назовите 3-5 архитекторов, на которых вы равняетесь (необязательно конкуренты)
- [ ] Какие сайты архитекторов вам нравятся и почему?
- [ ] Есть ли сайты, которые категорически не нравятся? Почему?

## 3. Услуги

### Перечень услуг (отметьте актуальные)
- [ ] Архитектурное проектирование частных домов
- [ ] Архитектурное проектирование коммерческих объектов
- [ ] Дизайн интерьеров (жилых)
- [ ] Дизайн интерьеров (коммерческих — рестораны, офисы, отели)
- [ ] Ландшафтный дизайн / благоустройство
- [ ] Концептуальное проектирование
- [ ] Реконструкция и реставрация
- [ ] BIM-моделирование
- [ ] Авторский надзор за строительством
- [ ] Подбор материалов и комплектация
- [ ] Консультации (разовые)
- [ ] Мастер-планирование (посёлки, кварталы)
- [ ] Другое: ___________________________

### Структура
- [ ] Какие услуги являются основными (приносят больше всего дохода)?
- [ ] Какие услуги хотите продвигать в первую очередь?
- [ ] Работаете один или привлекаете команду / субподрядчиков?
- [ ] Если есть команда — кто в неё входит? (инженеры, дизайнеры, визуализаторы)

### Ценообразование
- [ ] Указывать ли цены на сайте?
- [ ] Формат цен: точные / «от ... тенге за м²» / по запросу
- [ ] Есть ли пакетные предложения? (например: «Эскиз + Проект + Надзор»)
- [ ] Этапы работы и что входит в каждый этап

## 4. Портфолио и проекты

### Объём и структура
- [ ] Сколько проектов планируете показать? (5-10 / 10-20 / более 20)
- [ ] Категории проектов: по типу (дома / интерьеры / общественные) или по статусу (реализованные / в процессе / концепции)?
- [ ] Нужна ли фильтрация / сортировка проектов?
- [ ] Нужна ли отдельная детальная страница для каждого проекта?
- [ ] Хотите ли выделять «Проект месяца» или «Избранные»?

### Информация о каждом проекте
- [ ] Название проекта
- [ ] Описание / концепция
- [ ] Местоположение объекта
- [ ] Площадь
- [ ] Год проектирования / реализации
- [ ] Статус (реализован / в процессе / концепция)
- [ ] Бюджет проекта (показывать?)
- [ ] Использованные материалы и технологии
- [ ] Отзыв заказчика (если есть)
- [ ] Другое: ___________________________

### Визуальные материалы
- [ ] Есть ли профессиональные фотографии реализованных объектов?
- [ ] Есть ли 3D-визуализации?
- [ ] Есть ли видео (облёты, таймлапсы строительства, влоги)?
- [ ] Есть ли чертежи, планы, разрезы для публикации?
- [ ] Нужна ли профессиональная фотосъёмка объектов?
- [ ] Есть ли «процессные» фото (стройка, эскизы от руки, макеты)?

## 5. Контент и тексты

### Готовность
- [ ] Есть ли готовые тексты для сайта?
- [ ] Кто будет писать тексты? (сам / копирайтер / нужна помощь)
- [ ] Есть ли профессиональные фото вас лично? (портрет, за работой, на объекте)
- [ ] Есть ли готовая биография?

### Блог / Публикации
- [ ] Нужен ли раздел «Блог» или «Заметки архитектора»?
- [ ] Темы: процесс работы / тренды / разбор проектов / стройка / путешествия
- [ ] Как часто планируете публиковать?
- [ ] Есть ли публикации в СМИ, которые можно разместить?
- [ ] Ведёте ли YouTube / подкаст?

### Языковые версии
- [ ] На каких языках нужен сайт? (казахский / русский / английский / другие)
- [ ] Нужен ли переключатель языков?
- [ ] Есть ли готовые переводы?

## 6. Дизайн и визуал

### Референсы
- [ ] 3-5 сайтов, которые нравятся визуально (любая тематика)
- [ ] 3-5 сайтов архитекторов / дизайнеров, которые нравятся
- [ ] Что именно нравится? (навигация / анимации / подача портфолио / типографика / атмосфера)

### Фирменный стиль
- [ ] Есть ли личный логотип? В каких форматах?
- [ ] Есть ли брендбук / гайдлайн?
- [ ] Фирменные цвета (HEX-коды, если есть)
- [ ] Фирменные шрифты
- [ ] Другие элементы стиля (паттерны, подпись, монограмма)

### Предпочтения
- [ ] Цветовая гамма: светлая / тёмная / нейтральная
- [ ] Стиль: минимализм / брутализм / классика / современный / экспериментальный
- [ ] Фотостиль: светлые / тёмные, с людьми / без, детали / общие планы
- [ ] Важны ли анимации и интерактивные элементы?
- [ ] Нужен ли «вау-эффект» или спокойная элегантность?
