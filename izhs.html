Этапы работы

Этап 01

Знакомство

Место для текста. Встреча с заказчиком, обсуждение пожеланий, осмотр участка и анализ условий.

Этап 02

Эскизный проект

Место для текста. Разработка планировочных решений, фасадов и 3D-визуализации будущего дома.

Этап 03

Рабочий проект

Место для текста. Подготовка полного комплекта рабочей документации для строительства.

Этап 04

Согласование

Место для текста. Получение всех необходимых разрешений и прохождение экспертизы проекта.

Этап 05

Авторский надзор

Место для текста. Сопровождение строительства, контроль качества и соответствия проекту.
